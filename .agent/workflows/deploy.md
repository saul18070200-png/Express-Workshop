description: Cómo desplegar la landing page de NOVAX a internet
---

Para exponer tu página al mundo, la opción más sencilla y profesional para un sitio estático como este es **Netlify** o **Vercel**. Aquí tienes los pasos:

### Opción 1: Netlify (La más rápida)
1. Entra en [Netlify Drop](https://app.netlify.com/drop).
2. Arrastra y suelta la carpeta `CasaSoftware` directamente en el recuadro.
3. ¡Listo! Te dará una URL pública al instante.

### Opción 2: GitHub Pages (Para control de versiones)
1. Crea un repositorio en GitHub.
2. Sube tus archivos (`index.html`, `css/`, `js/`, `assets/`).
3. En la configuración del repo (Settings > Pages), selecciona la rama `main` y guarda.
4. GitHub te dará una URL tipo `usuario.github.io/repositorio`.

### Opción 3: Vercel
1. Instala la CLI de Vercel o conecta tu cuenta de GitHub en [vercel.com](https://vercel.com).
2. Importa el proyecto y se desplegará automáticamente.

**Recomendación:** Empieza con **Netlify Drop** para verla online en segundos.
